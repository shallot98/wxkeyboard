# WeType Vertical Swipe Toggle - 更新日志

## v1.0.1 (当前版本)

### 重大变更
- **上下滑动现在只切换中英文键盘**：任何位置（除了顶部任务栏）的上下滑动都只会在中英文键盘之间切换，不再根据区域触发数字/符号键盘
- **手势优先级提升至最高**：设置 `cancelsTouchesInView = YES`，确保覆盖微信输入法原有的滑动功能
- **日志系统改进**：在发布版本中也启用日志功能，方便用户反馈问题

### 详细改动
1. 移除了区域检测的键盘切换逻辑（数字键盘、符号键盘）
2. 所有检测到的垂直滑动手势都强制触发中英文切换
3. 增强日志输出，添加版本标识 `[v1.0.1]`
4. 所有日志消息都包含详细的手势和状态信息

### 日志位置
- 主日志：`/var/mobile/Library/Preferences/wxkeyboard.log`
- 备份日志：`/var/mobile/Library/Preferences/wxkeyboard.log.1`
- 系统日志：可通过 `Console.app` 查看，搜索 `[wxkeyboard]`

### 使用方法
1. 在任何支持微信输入法的应用中打开键盘
2. 在键盘区域（除顶部任务栏外）进行上下滑动
3. 键盘将在中文和英文之间切换
4. 查看日志文件获取详细调试信息

### 技术细节
- 手势识别器优先级：最高（覆盖原有功能）
- 顶部任务栏：不响应滑动（避免影响候选词和工具栏）
- 滑动阈值：默认 25 像素（可配置）
- 支持架构：arm64, arm64e
- 兼容性：rootless 越狱
